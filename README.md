# AdvocateHub

A simple content tracking tool built for developer advocates. Keep all your blog posts, videos, workshops, and social content organized in one place. Track what you've created, measure performance, and never lose sight of your content again.

## ✨ Features

- **Smart Content Capture**: Paste any URL and automatically detect platform and content type
- **Universal Search**: Find content across titles, descriptions, tags, platforms, and content types
- **Intelligent Unfurling**: Extract metadata (titles, descriptions, thumbnails) from URLs
- **Flexible Organization**: Categorize content into 11+ content types with custom tagging
- **Real-time Updates**: Instant UI updates with reactive state management
- **Local Storage**: All data persists locally in your browser
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## 🔍 Search Capabilities

The search feature allows you to quickly find content by searching across:
- Content titles and descriptions
- User-added tags
- Platform names (YouTube, Medium, Dev.to, etc.)
- Content types (blog, video, workshop, etc.)

Search supports:
- Multi-word queries (all terms must match)
- Partial word matching
- Case-insensitive search
- Real-time highlighting of search terms
